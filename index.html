Holiday

Input the country abbreviation and the year you want to search for public holidays and click the button to wait the system display the holiday list

(Input format: CN, US, UK, ...; For example: CN represents China)

(For example: 2020, 2021, ...)

Weather

Input the city and date you want to search for the recent weather and click the button to wait the system display the output list

(For example: Suzhou, Beijing, Shanghai, ...)

(For example: "JS" for Jiangsu (in China); "NY" for New York (in USA))

(Input format: CHN, USA, UK, ...; For example: CHN represents China)

(Input format: YYYY-MM-DD; For example: 2020-01-01)

Rental

Input the city you want to search for rental information and click the button to wait the system to display the output list

(For example: Beijing, Shanghai, Shenzhen...)
